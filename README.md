# Robot_quadrupede
Code Arduino implémentant les phases d'avance et de rotation d'un robot quadrupède à partir de la trajectoire générée par cinématique inverse (https://github.com/KLB-F/ProjetRobotique/tree/main).

- **rotation.ino** : le robot fait une rotation sur lui-même  
- **sprint.ino** : le robot parcours 3m en ligne droite le plus rapidement possible  
- **avance_et_rotation.ino** : fonctions de base permettant au robot de se déplacer dans tout l'espace
